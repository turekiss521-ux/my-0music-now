# 私人音乐播放器

基于免费音乐 API 的简单在线播放器，部署在 Cloudflare Pages。

## 部署
1. Fork 本仓库
2. Cloudflare Pages → Connect Git → 部署
3. 登录：admin / 123456

## 功能
- 搜索多源音乐（网易、QQ、酷狗等）
- 播放列表、歌词同步、封面显示
- 响应式设计，手机友好
